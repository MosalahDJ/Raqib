Raqib 
Overview

Raqib is a mobile application designed to support trade compliance and consumer protection by enabling users to report commercial violations and monitor their report status.
The application provides structured access to trade-related regulations and improves communication between citizens and trade authorities.

Core Features

Submit trade violation reports

Track report status and history

Access trade laws and regulations

User authentication

Clean, intuitive mobile UI

Technology Stack

Flutter & Dart

Bloc / Cubit (State Management)

Supabase (Authentication & Backend)

RESTful APIs

Architecture

The application follows a clean, scalable architecture with clear separation of concerns:

Data layer – API services and repositories

Logic layer – Business logic using Cubit

Presentation layer – Screens and reusable UI components

Requirements

Flutter SDK 3.16.0+

Dart SDK 3.2.0+

Android SDK 21+ (Android 5.0+)

iOS 12.0+
